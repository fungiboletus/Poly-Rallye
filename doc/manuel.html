Poly-Rallye

Public concerné : Tout public ; Pensé avant tout pour les non voyants

Note d'utilisation : Ce Jeux utilise Java et VocalyzeSIVOX

Poly'Rallye est un jeu vidéo de rallye pour les personnes non voyantes. Ce projet est réalisé dans le cadre du projet DeViNT 2011.

La conduite est basée sur l'environnement sonore et les instructions du copilote.

De nombreux championnats sont disponibles, après obtention d'un permis de conduire. Une grande collection de voitures de rallye est présente pour faire varier la difficulté et les plaisirs.

Exécution :

Le jeu s'execute en utilisant execution.bat

Déroulement du jeu

Le joueur commence dans le menu principal. Dans ce menu, il peut choisir 4 options :

  • Mode arcade - Ce jeu permet au joueur de lancer une course simple après avoir choisi sa voiture et le circuit
  • Mode Carrière - Ce mode permet au joueur d'acheter des voitures puis de concourir dans des championnats afin de gagner d'autres voitures
  • Permis - Ce mode permet au joueur de se familiariser à la conduite dans poly-rallye. Il est recommandé de ne pas bouger la voiture tant que le tutoriel audio ne s'est pas terminé
  • Configuration Joueur - Polyrallye permet de sauvegarder plusieurs joueurs. Ce menu permet de changer de joueur ou d'effacer la progression du joueur dans le mode carrière

Raccourcis clavier

  • A tout instant
    1. F2 - permet de consulter l'aide
    2. d - Affiche le mode debug (Affichage de tout ce qu'il se passe dans le jeux, non visible par un MV)
  • Dans un menu
    1. F1 - permet de consulter le but du jeu
    2. flèche haut - permet de naviguer dans un menu
    3. flèche bas - permet de naviguer dans un menu
    4. Entrée - Valide le menu sélectionné
    5. Echap - retourne au menu précedent
  • Dans une course
    1. flèche haut - permet d'accélérer
    2. flèche bas - permet de freiner
    3. flèche gauche - permet de tourner à gauche
    4. flèche droite - permet de tourner à droite
    5. Echap - Quitte la course
    6. Espace - Joue le klaxon
    7. a - Active/Desactive la boite de vitesse manuelle
    8. Maj gauche - Passe la vitesse
    9. Ctrl gauche - Rétrograde la vitesse
    10. c - Le copilote fiat la discussion
    11. e - Allume / Eteint la radio
    12. r - Change la station de radio
    13. t - Augmente le volume de la radio
    14. y - Diminue le volume de la radio
