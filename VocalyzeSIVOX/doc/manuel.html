Le laboratoire VocalyzeSIVOX

Ce logiciel a été développé en grande partie sous la forme de projets de deuxième année du département Sciences Informatiques de l'EPU de Nice Sophia Antipolis, sous l'encadrement de Hélène Collavizza et Jean-Paul Stromboni pour servir les projets DeViNT (Déficients Visuels et Nouvelles Technologies) depuis 2004 et jusqu'en 2007.

Il y a deux raisons à sa présence sur le CD:

  1. Tous les projets DeViNT qui utilisent une synthèse vocale la trouvent dans un répertoire nommé VocalyzeSIVOX dans le répertoire LesLogiciels qui contient tous les projets :
    1. dans VocalyzeSIVOX/bin/SIVOX.jar, toutes les méthodes et variables de la synthèse vocale, regroupées dans uen API nommée SIVOXDevint.java
    2. dans /ressources/donneesMbrola, ils trouvent les voix MBROLA dans fr1/fr1, fr2/fr2, etc ... ainsi que le moteur Mbrola.exe pour Windows (et l'équivalent pour Unix) qui est utilisé par SI_VOX.jar.
    3. Les projets doivent toutefois conserver dans le dossier ressources un sous-dossier donneesMbrola qui contient :
      1. les règles de prononciation dans le sous-dossier rules, cf. par exemple le fichier regles.txt
      2. les fichiers de phonèmes .pho et les fichiers wave .wav créés par Mbrola
  2. Un utilisateur voyant du CD pourra expérimenter la synthèse vocale de textes tapés ou lus dans des fichiers textes en exécutant VocalyzeSIVOX comme un autre projet.

Utilisation de SI_VOX

utiliser l'aide associée à l'interface de VocalyzeSIVOX

Conditions nécessaires

le logiciel utilise java JRE 1.6 et sa bibliothèque SWT pour l'interface
